
This is a Python-based text editor that suggests words as you type based on existing corpora

Cassandra Xia
xiac@mit.edu

=================
 REQUIREMENTS 
=================
Python 2.7
PyQt4
nltk

nltk's gutenberg and brown corpora, to get these corpora:
>>> import nltk
>>> nltk.download()

=================
 TO RUN
=================
To edit corpora:
config.py

To run:
python gui.py
